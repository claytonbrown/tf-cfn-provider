# Terraform::OneAndOne::Vpn

Manages a VPN on 1&1

## Properties

`Datacenter` - (Optional) Location of desired 1and1 datacenter. Can be `DE`, `GB`, `US` or `ES`.

`Name` - (Required) The name of the VPN.

`Description` - (Optional).

`DownloadPath` - (Optional).

`FileName` - (Optional).
