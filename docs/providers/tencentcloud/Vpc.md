## Return Values

### Fn::GetAtt

`Id` - The ID of the VPC.

`Name` - The name for the VPC.

`CidrBlock` - The CIDR block of the VPC.

`IsDefault` - Whether or not the default VPC.

`IsMulticast` - Whether or not the VPC has Multicast support.
