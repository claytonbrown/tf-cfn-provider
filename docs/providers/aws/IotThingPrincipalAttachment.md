# Terraform::AWS::IotThingPrincipalAttachment

Attaches Principal to AWS IoT Thing.

## Properties

`Principal` - (Required) The AWS IoT Certificate ARN or Amazon Cognito Identity ID.

`Thing` - (Required) The name of the thing.
