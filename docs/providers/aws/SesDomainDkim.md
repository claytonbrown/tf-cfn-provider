# Terraform::AWS::SesDomainDkim

Provides an SES domain DKIM generation resource.

Domain ownership needs to be confirmed first using [ses_domain_identity Resource](/docs/providers/aws/r/ses_domain_identity.html)

## Return Values

### Fn::GetAtt
