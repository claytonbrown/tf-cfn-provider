# Terraform::GitHub::UserSshKey

Provides a GitHub user's SSH key resource.

This resource allows you to add/remove SSH keys from your user account.

## Properties

TBC

## Return Values

### Fn::GetAtt

`Id` - The ID of the SSH key.

`Url` - The URL of the SSH key.
